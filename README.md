# GuessNumber
# Игра "Угадай число"

Простая консольная игра на Python.

## Возможности
- Три уровня сложности (1-10, 1-100, 1-1000)
- Защита от ввода букв
- Проверка диапазона
- Счётчик попыток
- Дружелюбные сообщения

## Как запустить
1. Установи Python 3.x
2. Скачай файл `guess.py`
3. Запусти в терминале: `python guess.py`
